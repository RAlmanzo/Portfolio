SniffHikes Web app

Description:

This web application is designed for people who enjoy hiking in nature with their dogs. After registering, users can create and share hiking events and routes, as well as leave comments on posts from others. The backend is built using an ASP.NET Core Web API, with API documentation provided via Swagger for structured and accessible endpoint reference. The application also includes an admin section that allows management of users, events, routes, and comments to ensure smooth moderation and control. This project combines frontend and backend development, and taught me how to implement user authentication/authorization, CRUD operations, and API integration within the .NET ecosystem.

Tech Stack:

Frontend: ASP.NET Core MVC, HTML5, CSS, Bootstrap, Vue2

Backend: C#, ASP.NET Core Web API (REST), Swagger, Microsoft Identity, EF Core (ORM), Microsoft SQL Server, Clean Architecture, Anotation Validation, Xunit

SniffHikes Mobile app (Android/ Windows)

Description:

As an extension of my hiking web application, I developed a cross-platform mobile app using Xamarin.Forms targeting Android and Windows devices. This app enables registered users to share and explore hiking events and routes tailored for dog owners. Users can post comments, view detailed addresses, and access route navigation via maps or apps like Waze. The app also allows users to register for events, manage their profiles—including adding their dogs—and discover hiking zones in Belgium specifically suitable for dogs, with the ability to share experiences through comments. A key feature(wich is still in progress) is the ability to record their own hikes, share these routes with the community, and receive feedback. Additionally, the app includes an admin section for managing users, events, routes, and content, ensuring smooth operation and community moderation. This project demonstrates my skills in mobile development, cross-platform design, user interaction, and backend integration.

Tech Stack:

Frontend: C#, Xamarin Forms (Android, Windows), MVVM, XAML, Databinding, Xunit

Backend: C# GraphQL API (...in progress), Hotchocolate .NET, Firebase Cloud DB
